# starting-point

A starting point for my projects based off of the yeoman webapp generator with assemble added and some fine tuning of the basic HTML 5 Boilerplate.

## requirements
1.  node
2.  grunt
3.  bower
4.  sass

## installation
1.  clone this repository
2.  run `npm install`
3.  run `bower install`
4.  run `grunt serve`
5.  done!
